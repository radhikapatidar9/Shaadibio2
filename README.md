# ShaadiBio

ShaadiBio is a web application that allows users to easily create and download beautiful matrimonial biodata. Users can select templates, enter personal details, preview their biodata, and download it as a PDF.

---

## Live Demo

https://shaadibio2.onrender.com
  
---

## Features

- User authentication (Signup/Login)
- Create matrimonial biodata
- Multiple biodata templates
- Preview biodata before downloading
- Download biodata as PDF
- Responsive UI

---

## Installation

### Clone the repository
```bash
git clone https://github.com/radhikapatidar9/Shaadibio.git
cd Shaadibio
```

### Backend setup
```bash
cd backend
npm install
npm start
```

### Frontend setup
```bash
cd frontend
npm install
npm run dev
```
